Để kết nối vào một máy chủ MySQL đang chạy trong một container Docker bằng Sequel Ace, bạn cần biết địa chỉ IP của container MySQL và cổng mà MySQL đang lắng nghe trên. Đây là các bước bạn có thể thực hiện:

Lấy địa chỉ IP của container MySQL:

Chạy lệnh sau trong terminal để lấy danh sách các container Docker đang chạy:
Copy code
docker ps
Tìm container MySQL trong danh sách và ghi nhớ tên hoặc ID của nó.
Sau đó, chạy lệnh sau để lấy địa chỉ IP của container MySQL:
arduino
Copy code
docker inspect <container_name_or_id> | grep IPAddress
Ghi nhớ địa chỉ IP này cho bước tiếp theo.
Kết nối bằng Sequel Ace:

Mở Sequel Ace trên máy tính của bạn.
Nhấp vào nút "New Connection" để tạo kết nối mới.
Điền các thông tin sau:
Name: Tên cho kết nối của bạn (ví dụ: MySQL Docker).
Host: Địa chỉ IP của container MySQL (bước 1).
Port: Số cổng mà MySQL đang lắng nghe trên container (thường là 3306).
User: Tên người dùng MySQL.
Password: Mật khẩu của người dùng MySQL.
Sau đó, nhấp vào nút "Test Connection" để kiểm tra xem kết nối có thành công không.
Nếu kết nối thành công, bạn có thể nhấp vào nút "Connect" để kết nối vào cơ sở dữ liệu MySQL trong container Docker của bạn.
Lưu ý rằng trong một số trường hợp, bạn có thể cần xác định các thiết lập bổ sung như database name hoặc các thiết lập khác tùy thuộc vào cấu hình của container MySQL của bạn.

CHÚ Ý :(docker exec -it container_mysql_name mysql -u username -p)
+ username : là user name connect, nó nằm trong file .env
DB_CONNECTION=mysql
DB_HOST=mysql
DB_PORT=3306
DB_DATABASE=laravel10_vue3_cart
DB_USERNAME=hoanguyencoder
DB_PASSWORD=12345678
+ container_mysql_name : là name của container mysql có trong khi ta run docker lên

docker exec: Dùng để thực thi một lệnh trong một container Docker đang chạy.
-it: Chạy lệnh trong chế độ tương tác (interactive mode) và sử dụng terminal (TTY).
container_mysql_name: Tên của container Docker mà bạn muốn thực thi lệnh trong đó.
mysql: Là lệnh để thực thi MySQL client.
-u username: Sử dụng tài khoản người dùng MySQL với tên người dùng là "username".
-p: Được sử dụng để yêu cầu nhập mật khẩu sau khi kết nối đến server MySQL.

hoacode@HOACODEs-MacBook-Pro ~ % docker exec -it container_mysql_name mysql -u  -p
